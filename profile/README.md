## Common Questions and Solutions

- Application won't launch? Right-click the executable and select "Run as Administrator" to grant the low-level hardware access this CPU-Z tool requires for reading sensor data.
- Readings seem incorrect? Update to the CPU-Z latest version, as older releases lack support for recent processor generations and may display wrong specifications for new silicon.
